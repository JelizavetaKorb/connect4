# Connect4 Game

A multi-platform Connect4 game that can be played in **console** or **web**, supporting Human vs Human, Human vs AI, and AI vs AI. Features a smart AI with the **MinMax algorithm**, flexible storage options, and multiplayer gameplay with friends through join codes.

## Key Features

- **Gameplay Modes**
  - Human vs Human  
  - Human vs AI (MinMax AI)  
  - AI vs AI  

- **Flexible Storage**
  - Save, load, continue, and delete games  
  - Supports both **JSON file storage** and **Entity Framework database**  
  - Switch in a few lines between storage types without losing any game data  

- **Cross-Platform Play**
  - Start a game in the **console app** and continue in the **web app**, or vice versa  
  - Play with a friend in the web app using a **join code**  

- **Game Variants**
  - Classical Connect4  
  - ConnectX (customized board)
  - ConnectX-cylinder  

- **Web Features**
  - Multiplayer support with unlimited parallel games across different browsers/tabs  
  - Smooth interactive game interface
  - Turn-based gameplay with clear instructions

## Technologies Used

- C#, .NET  
- JavaScript
- HTML, CSS  
- Microsoft Entity Framework
- JSON file storage
